# letterpress

A terminal-first letter and card composer for beautiful printable messages.

## Status

This repository is being bootstrapped. Initial implementation work is tracked in GitHub Issues and will follow an issue -> branch -> PR workflow targeting `develop`.

## Development

Use the Make targets below for the local workflow:

- `make run` launches the current application entrypoint.
- `make test` runs the test suite.
- `make fmt` formats the codebase with `go fmt`.
- `make tidy` syncs module dependencies.

You can also inspect the root command with `go run ./... --help`.
